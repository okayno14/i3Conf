# Deploy

1. `git submodule update --init`
2. `./build.sh`
3. Set up `$monitor_primary`, `$monitor_secondary` in out/.config/i3/config
   `apply_conf.sh`
4. Copy config and install dependencies with `deploy.sh`

    ./build.sh && ./apply_conf.sh && ./deploy.sh get_deps
